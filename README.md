# 🌀 MazeBlitz

MazeBlitz is a modern Android Maze Game built using Kotlin + Jetpack Compose with FastAPI backend integration.  
Players solve randomly generated mazes, unlock levels, track scores, and compete on the leaderboard.

---

# ✨ Features

- 🎮 Dynamic Maze Generation
- 🚀 Multiple Difficulty Levels
- 🏆 Leaderboard System
- 📜 Player History Tracking
- 🔓 Level Unlock System
- ⏱️ Timer + Score Calculation
- 🌐 Android + Backend API Integration
- 📱 Modern Jetpack Compose UI

---

# 🛠️ Tech Stack

## Android
- Kotlin
- Jetpack Compose
- Navigation Compose
- Retrofit
- MVVM Architecture

## Backend
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
